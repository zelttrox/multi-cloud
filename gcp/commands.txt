Login
gcloud auth login

Make bucket
gsutil mb -p hardy-album-463309-i0 -c standard -l europe-west3 gs://multi-cloud-app

Copy all files to bucket
gsutil cp -r ./assets gs://multi-cloud-app

Set default website page
gsutil web set -m index.html gs://multi-cloud-app

Set files visibility to public
gsutil iam ch allUsers:objectViewer gs://multi-cloud-app
